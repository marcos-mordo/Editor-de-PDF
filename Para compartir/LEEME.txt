==============================================
  EDITOR DE PDF v0.8.0
  Alternativa libre a Adobe Acrobat
==============================================

(Confirmar version: Menu Ayuda -> Acerca de -> v0.8.0)


==============================================
 NOVEDADES DE ESTA VERSION
==============================================

 PROTECCION CON CONTRASENA REAL (AES-128)
   Antes solo marcaba metadatos. AHORA cifra de verdad
   con el estandar de seguridad de PDF (el mismo que
   Adobe Acrobat). El PDF pide la contraseña para
   abrirse en cualquier lector: Adobe, Chrome, Firefox.
   Verificado contra el lector de Mozilla (pdf.js).
   Menu Herramientas -> Proteger con contraseña.
   * Guarda bien la contraseña: sin ella el documento
     no se puede recuperar.

 REDUCIR TAMANO DEL PDF
   Menu Herramientas -> Reducir tamaño del PDF.
   - Optimizar (sin perdida): mantiene el texto.
   - Rasterizar: convierte paginas en imagen JPEG con
     control de calidad/resolucion. Reduce mucho en
     escaneos. Muestra el % de reduccion.

 RECORTAR PAGINAS (CROP)
   Menu Archivo / boton Recortar. Arrastra el area a
   mantener; aplica a la pagina actual o a todas.

 PROPIEDADES Y PRIVACIDAD
   Menu Archivo -> Propiedades del documento.
   Edita titulo/autor/etc. o pulsa "Eliminar todos los
   metadatos" para limpiar el documento.

 VISTAS Y PREFERENCIAS
   - Ajustar al ancho (Ctrl+1), a la pagina (Ctrl+2),
     tamaño real (Ctrl+0).
   - Preferencias (Ctrl+,): vista al abrir, panel, etc.
     Se recuerdan entre sesiones.
   - Atajos de teclado (Ctrl+/): lista completa.


==============================================
 EDITOR DE TEXTO (la herramienta principal)
==============================================

 Activa "Editor de texto", pasa el raton (se resalta la
 palabra editable) y haz click. Edita el texto ORIGINAL
 del PDF respetando fuente, tamaño, color y posicion,
 SIN poner cajas encima. Soporta fuentes CID/Type0 de
 Canva y plantillas de CV.

==============================================
 INSTALACION
==============================================

OPCION 1: "Editor de PDF - Instalador.exe" (doble click)
OPCION 2: "Editor de PDF - Portable.exe" (sin instalar)

Windows mostrara "SmartScreen" (app sin firma de pago).
Pulsa "Mas informacion" -> "Ejecutar de todas formas".
